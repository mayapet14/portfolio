RepairMasters App

The RepairMasters App is a mobile application that provides organizational support for mechanics. A categorized database allows mechanics to log required parts. documentation, pricing, and share data with customers. This product meets mechanics needs through organizational structure, simple UX design, and improves personal business processes. With the RepairMasters app, mechanics are given free administrative relief that saves around 5 hours weekly of administrative tasks.

Product Backlog Items: user profile for mechanics, a display page with a table containing cat part categorization, allows users to record final prices and create a projects overview page. Also, mechanics can categorize projects by type, input car/ part data, and automatically notify clients about timeframe based on note documentation. Additionally, mechanics can hierarchically organize jobs, document issues, and upload media. Lastly, mechanics can create a client list and profiles, and search the app for specific jobs and parts.

User Profile and Search
Users will have their full name displayed on the top of the page. There will also be a menu listing the app's available features. Users can use the search bar to look for specific information, where it will appear on a dropdown menu. Testing this item was successful and users were able to quickly and easily navigate the profile.
Project Page
Users can list all projects they are currently working on. Once the edit button is clicked, it will take users to a page where they can edit the progress bar, issues, and notes. At the bottom of there page, there is a button to save the edits. Users were able to successfully navigate the edit process.
